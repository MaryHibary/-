# Tourism Web Application

## Описание проекта

Данный проект разработан в рамках выполнения кейс-задания по дисциплине.

Тема проекта — «Туризм».

WEB-приложение позволяет просматривать информацию о странах, городах, типах туров, клиентах и оформленных заказах. Для хранения информации используется база данных MySQL.

## Используемые технологии

- Python
- Flask
- HTML
- CSS
- MySQL
- SQLAlchemy

## Структура базы данных

База данных включает пять таблиц:

- Countries — страны;
- Cities — города;
- Tour_Types — виды туров;
- Clients — клиенты;
- Orders — оформленные заказы.

Таблицы связаны между собой с помощью первичных и внешних ключей.

## Анализ WEB-приложений

В настоящее время WEB-приложения широко используются в различных организациях. Они позволяют работать через браузер без установки программы на компьютер, обеспечивают доступ к данным из любого места и упрощают совместную работу пользователей.

Примеры популярных WEB-приложений:

### Bitrix24

Используется для управления компанией, работы с клиентами, постановки задач и хранения документов.

Преимущества:
- работа через браузер;
- совместная работа сотрудников;
- удобное управление задачами.

### Trello

Предназначено для управления проектами и задачами.

Преимущества:
- простой интерфейс;
- наглядное отображение задач;
- возможность совместной работы.

### Google Docs

Используется для создания и совместного редактирования документов.

Преимущества:
- автоматическое сохранение;
- одновременная работа нескольких пользователей;
- хранение документов в облаке.

## Вывод

WEB-приложения позволяют автоматизировать работу организаций, обеспечивают удобный доступ к информации и повышают эффективность совместной работы.

В рамках данного проекта было разработано WEB-приложение «Туризм», взаимодействующее с базой данных MySQL.
